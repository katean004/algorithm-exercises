Big O Notation

Big O Notation helps to analyze performance of algorithms by evaluating the time complexity and space complexity of algorithms.

Different Big O Types

Below are the Big O types from most efficient to least efficient in descending order:

  • O(1) is constant
  • O(logn) is logarithmic
  • O(n) is linear
  • O(nlogn)
  • O(n^2) is quadratic

Notes

  • Most primitive data are constant space, such as booleans, numbers, undefined, and null.
  • Strings require O(n) space where n is the string length.

Big O for Objects

  • Insertion: O(1)
  • Removal: O(1)
  • Search: O(n)
  • Access: O(1)

Big O for Arrays

  • Insertion: depends if it's from the beginning or end.
  • Removal: depends if it's from the beginning or end.
  • Search: O(n)
  • Access: O(1)
